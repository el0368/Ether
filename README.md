## üõ†Ô∏è Getting Started

### Prerequisites
*   Elixir 1.15+ & Erlang/OTP 26+
*   Node.js 20+
*   Git

### Installation
1.  Clone the repository.
2.  Install dependencies:
    ```bash
    mix setup
    ```

### Running the Desktop App (Recommended)
We provide a helper script to launch the full environment:
```powershell
.\start_desktop.bat
```
This will compile assets, start the server, and open the Aether IDE window.

### Web-Only Development
If you prefer developing in the browser:
```bash
./start_dev.bat
```
Then visit: `http://localhost:4000`

## üß† Architecture

Aether allows "One Runtime" simplicity. The Elixir BEAM VM handles everything:
-   HTTP Server (Bandit)
-   WebSocket Channels (Phoenix)
-   Background Agents (GenServer)
-   GUI (wxWidgets)

No Rust/Zig NIFs are required for the core logic, ensuring strict stability and introspection capabilities.
